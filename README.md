# 🗺️ Travel Itinerary Generator

> 让 AI 帮你做旅行行程单：从零规划到完整攻略

---

## 这是什么？

一个自用的 AI Skill，支持两种场景：

1. **从零规划** — 还没订机票酒店？通过问答了解你的偏好，推荐住哪+怎么玩
2. **整理已有信息** — 已经订好了？把截图丢给它，自动整理成可打印的行程单

---

## 痛点

- 机票在航旅纵横、酒店在爱彼迎、景点在小红书收藏夹
- 同行的朋友问"几点的飞机""酒店地址发一下"，每次都要翻半天
- 想做一份好看的行程单，排版画表格花了一下午
- **还没订酒店，不知道住哪个区域最方便**
- **收藏了一堆景点，不知道怎么取舍和安排**

## 如何使用

### 场景 A：从零规划

对 AI 说：
> "我想去布拉格，帮我规划一下"

AI 会问你时间、人数、偏好、预算，然后推荐住宿区域 + 生成行程单。

### 场景 B：整理已有信息

对 AI 说：
> "帮我整理一下釜山旅行行程单"

然后发送机票截图、酒店截图、想去的景点，AI 自动整合成完整行程单。
